Understanding complex DAGs can be difficult. HyperNodes provides built-in visualization tools to help you see the structure of your pipeline.

## The `.visualize()` Method

You can generate a Graphviz diagram of any pipeline using the `.visualize()` method.

```python
pipeline.visualize()
```

- **Jupyter Notebooks**: Renders the SVG directly in the cell.
- **Scripts**: Returns a `graphviz.Digraph` object.

### Saving to File

```python
pipeline.visualize(filename="my_pipeline.svg")
```

## Customization

### Orientation

Change the layout direction with `orient`:

```python
pipeline.visualize(orient="LR")  # Left-to-Right (default is Top-Bottom)
```

### Nesting Depth

Control how deep to expand nested pipelines.

- `depth=1`: Show top-level nodes and collapsed sub-pipelines.
- `depth=None`: Fully expand all nested pipelines.

```python
pipeline.visualize(depth=None)
```

### Styles

HyperNodes comes with visual styles.

```python
pipeline.visualize(style="modern")
```

## Legend

Enable a legend to explain node shapes and colors.

```python
pipeline.visualize(show_legend=True)
```
