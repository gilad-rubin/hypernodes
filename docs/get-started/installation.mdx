---
title: Installation
description: 'Get up and running with HyperNodes.'
---

<Steps>
  <Step title="Install the package">
    <CodeGroup>
      ```bash uv
      uv add hypernodes
      ```

      ```bash pip
      pip install hypernodes
      ```
    </CodeGroup>
  </Step>

  <Step title="Optional Dependencies">
    <CodeGroup>
      ```bash Daft (Recommended)
      uv add "hypernodes[daft]"
      # or
      pip install "hypernodes[daft]"
      ```

      ```bash Modal
      uv add "hypernodes[modal]"
      # or
      pip install "hypernodes[modal]"
      ```

      ```bash Telemetry
      uv add "hypernodes[telemetry]"
      # or
      pip install "hypernodes[telemetry]"
      ```
    </CodeGroup>
  </Step>
</Steps>
