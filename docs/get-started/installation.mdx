---
title: Installation
description: 'Get up and running with HyperNodes.'
---

<Steps>
  <Step title="Install HyperNodes">
    Requires Python 3.12+.

    ```bash
    pip install hypernodes
    ```
  </Step>
  <Step title="Optional Dependencies">
    Install extras for specific features:

    <CodeGroup>
    ```bash Visualization
    pip install "hypernodes[viz]"
    ```

    ```bash Modal
    pip install "hypernodes[modal]"
    ```

    ```bash Telemetry
    pip install "hypernodes[telemetry]"
    ```
    </CodeGroup>
  </Step>
</Steps>
