## Basic Pipeline

Create a simple pipeline that cleans text and counts words.

<Steps>
  <Step title="Define Nodes">
    Functions become nodes with the `@node` decorator.

    ```python
    from hypernodes import Pipeline, node

    @node(output_name="cleaned_text")
    def clean_text(passage: str) -> str:
        return passage.strip().lower()

    @node(output_name="word_count")
    def count_words(cleaned_text: str) -> int:
        return len(cleaned_text.split())
    ```
  </Step>
  <Step title="Build Pipeline">
    Dependencies are automatically resolved based on parameter names.

    ```python
    pipeline = Pipeline(nodes=[clean_text, count_words])
    ```
  </Step>
  <Step title="Run">
    Execute with a single input.

    ```python
    result = pipeline(passage="  Hello World  ")
    print(result)
    # {'cleaned_text': 'hello world', 'word_count': 2}
    ```
  </Step>
</Steps>

## Scale with `.map()`

Process multiple inputs without changing your pipeline code.

```python
results = pipeline.map(
    inputs={"passage": ["  Hello  ", "  World  "]},
    map_over="passage"
)
```

## Enable Caching

Add `DiskCache` to avoid re-running expensive computations.

```python
from hypernodes import SequentialEngine, DiskCache

engine = SequentialEngine(cache=DiskCache(path=".cache"))
pipeline = Pipeline(nodes=[clean_text, count_words], engine=engine)

# First run: executes
pipeline(passage="Hello World")

# Second run: instant cache hit!
pipeline(passage="Hello World")
```
